# лабораторна №21

консольна програма на c++, що рахує вартість абонементу спортзалу. використано патерн strategy для різних типів проходу: morning, day, full, student. вбудована перевірка вводу користувача

## що робить
- обирає тип абонементу та години
- за потреби додає додаткові послуги: сауна або басейн
- для кожного типу застосовує власну ставку та знижку
- валідовує всі введення; фабрика генерує помилку для невідомого типу

## логіка ціни
- morning: знижена ставка та знижка на додаткові послуги
- day: базові ставки без знижки
- full: вища ставка, але є знижка
- student: мінімальні ставки з найбільшою знижкою
